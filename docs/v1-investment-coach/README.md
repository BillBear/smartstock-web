# V1 投资教练方案包

本目录用于落地 `AI 自动选股教练` V1，目标是把 SmartStock 从"信息展示"升级为"可执行决策"。

## 文档清单

1. `01-feature-breakdown.md`
- 功能拆解
- 业务闭环与角色价值
- 版本范围与验收标准

2. `02-data-model.md`
- 数据模型与表结构
- 主键/索引/分区建议
- 关键口径定义

3. `03-api-contract.md`
- 后端接口契约
- 请求/响应示例
- 错误码与状态流

4. `04-wireframes.md`
- 页面线框（低保真）
- 关键交互
- 与现有路由映射

## 路由映射（现有前端）

- `/dashboard` -> 市场全景（市场环境）
- `/smart-screen` -> 今日可投（自动选股）
- `/backtest` -> 策略回溯（证据与实验）

## 建议开发顺序

1. 先完成数据层和离线任务（特征、概率、推荐、回溯）
2. 再开放查询接口（今日可投、个股决策、策略证据）
3. 最后改造页面与交互（先让用户能执行，再优化视觉）
